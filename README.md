Developed an AI-driven multilingual conversational banking assistant that enables users to perform banking operations through voice commands over phone calls. Integrated speech-to-text (AssemblyAI), intent recognition (OpenAI), and text-to-speech (ElevenLabs) with a Flask backend and Exotel VoIP system to handle real-time interactions such as balance inquiries and transactions. Designed the system to support multiple languages and improve accessibility for rural, elderly, and visually impaired users, with a scalable architecture for seamless banking automation
