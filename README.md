The proposed system is a multilingual voice-based banking interface that allows users to perform banking operations using simple voice commands in regional languages. Using advanced speech recognition and natural language processing, the system understands user queries, detects their intent, and securely executes actions such as balance inquiries, transaction checks, and fund transfers. The platform also provides real-time voice responses for a natural conversational experience. In addition, the AI performs emotion analysis on the user’s voice to detect stress, confusion, or urgency, allowing the system to respond more intelligently and provide better assistance. This solution improves accessibility for elderly users, rural populations, and individuals with limited digital literacy.
